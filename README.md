## 🚀 Sobre o Projeto

**CPU Chronicles** é uma aplicação web interativa (Landing Page) desenvolvida para contar a história e a evolução das Unidades Centrais de Processamento (CPUs) ao longo de seis grandes gerações. Com um design tech/futurista, o projeto utiliza animações fluidas e elementos visuais imersivos para explicar marcos históricos e conceitos tecnológicos de forma simples e engajadora.

## ✨ Funcionalidades

- **Interface Imersiva:** Cursor dinâmico personalizado e um fundo animado com partículas interativas que se conectam, construído com HTML5 Canvas.
- **Timeline Histórica:** Navegação cronológica cobrindo 6 eras da computação (Válvulas, Transistores, Circuitos Integrados, Microprocessadores, Multicore e IA).
- **Infográficos Animados:** Gráficos de barra e linhas (como a Lei de Moore) que se animam de forma fluida ao entrar na tela, proporcionando uma ótima experiência de scroll.
- **Modais Ricos em Detalhes:** Cartões expansíveis contendo fotos, especificações técnicas detalhadas, contexto de época e o impacto global de cada geração.
- **Design Responsivo:** A interface se adapta perfeitamente a smartphones, tablets e desktops.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando tecnologias web nativas (Vanilla), com foco em performance e animações sem dependências externas:

- **HTML5:** Semântica e estruturação do conteúdo em seções.
- **CSS3:** Estilização moderna utilizando Variáveis (Custom Properties), Flexbox, CSS Grid, Media Queries e Keyframes.
- **JavaScript (Vanilla):**
  - `Canvas API`: Para o renderizador da rede neural (partículas ao fundo).
  - `IntersectionObserver`: Gatilho de animações e carregamentos ativados pelo scroll.
  - `requestAnimationFrame`: Responsável por transições suaves em contadores numéricos (Hero Section) e no rastro do cursor.

## ⚙️ Como executar o projeto

Como é um projeto puramente Front-end sem frameworks ou processos de build complexos, rodá-lo localmente é extremamente simples:

1. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/CPU_Chronicles_-_Evolu-o_dos_Processadores.git
   ```
2. Acesse a pasta do projeto no seu terminal:
   ```bash
   cd CPU_Chronicles_-_Evolu-o_dos_Processadores
   ```
3. Dê um duplo clique no arquivo `index.html` para abri-lo em seu navegador de preferência, ou utilize a extensão **Live Server** no VSCode para uma experiência de desenvolvimento em tempo real.
